Add application

Register an app from its manifest. It starts registered (hidden), then you activate it to appear in the menu.

✓ Manifest
→
2 Registered
→
3 Activated
immutable · ^[a-z0-9-]+$
FULL entry URL (…/remoteEntry.js)
registered → activate → activated → appears in the application menu

POST /api/v1/app-registry/apps

{
  "manifest": {
    "manifestVersion": "1",
    "slug": "market",
    "name": "Market",
    "menuLabel": "Market",
    "mode": "portal",
    "integration": {
      "type": "module-federation",
      "remoteEntry": "https://market.example.com/remoteEntry.js",
      "scope": "marketApp",
      "module": "./MarketApp"
    },
    "chrome": { "menu": "host" },
    "visibility": "organization"
  }
}

→ 201 · then POST …/market/activate

"status": "activated"
